# Design

## Theme

절제된 모던에 강한 컬러 포인트. 차가운 기 도는 near-white 베이스 위에 near-black 잉크, 단 하나의 대담한 버밀리언 액센트. 개발자 신뢰감(Linear/Vercel 계열의 정밀함)에 본인 색을 더한다. 다크가 아닌 라이트: 학습도구·연구 자료를 낮 시간 데스크톱·모바일에서 또렷이 읽는 맥락.

색 전략: Restrained(중립 + 액센트 한 점). 액센트는 표면의 10% 미만으로, 강조·링크·핵심 수치·호버에만.

## Color (OKLCH)

| Role | Token | Value |
|---|---|---|
| 배경 | `--bg` | `oklch(0.985 0.004 250)` 차가운 off-white (크림 아님) |
| 표면 | `--surface` | `oklch(1 0 0)` |
| 패널 | `--panel` | `oklch(0.968 0.006 255)` |
| 경계 | `--border` | `oklch(0.91 0.008 255)` |
| 강한 경계 | `--border-strong` | `oklch(0.84 0.01 255)` |
| 본문 잉크 | `--ink` | `oklch(0.23 0.022 265)` |
| 2차 텍스트 | `--ink-2` | `oklch(0.40 0.02 265)` |
| 뮤트(라벨·대형만) | `--muted` | `oklch(0.52 0.018 265)` |
| 액센트(장식·대형) | `--accent` | `oklch(0.63 0.205 33)` 버밀리언 |
| 액센트(텍스트·링크) | `--accent-strong` | `oklch(0.52 0.18 32)` |
| 액센트 틴트 배경 | `--accent-tint` | `oklch(0.95 0.035 45)` |

대비 규칙: 본문은 `--ink`(약 14:1). 2차 텍스트 `--ink-2`. `--accent`는 대형/굵은/장식 전용, 텍스트로 쓸 땐 `--accent-strong`.

## Typography

- 본문·제목: **Pretendard Variable** (한국어 1차). 단일 패밀리를 무게/크기 대비로 운용(800~900 디스플레이, 700 제목, 440 본문).
- 기술 메타데이터(저장소 경로, 수치, 태그): **JetBrains Mono** (소량). 사용자가 실제 기술 도메인이므로 코스튬 아님.
- 스케일: `clamp()` 유동, 단계 간 ≥1.25. 히어로 `clamp(2.6rem, 7vw, 4.8rem)`, letter-spacing `-0.03em`, `text-wrap: balance`. 본문 줄길이 65~75ch.

## Components

- **헤더**: 슬림, 워드마크 + 섹션 앵커 + GitHub. 히어로 위 투명, 스크롤 시 미세 배경.
- **히어로**: 비대칭 2단(텍스트 + 분산-컴퓨팅 노드 SVG 모티프, 졸업논문 주제 은유). 액센트는 이름 마커·핵심 한 단어에만.
- **프로젝트("만든 것")**: 카드 그리드 금지. 대표 1개(featured, 큰 블록) + 나머지는 구분선 있는 에디토리얼 리스트(행). side-stripe·동일카드·01/02 번호 금지. 호버 시 액센트·미세 이동.
- **푸터**: 연락·소속, 차분.

## Motion

CSS 로드 진입(스태거 fade-rise), 호버 리프트/액센트. `prefers-reduced-motion: reduce`에서 즉시 표시(애니메이션 제거). 스크롤·IO로 가시성 게이트하지 않음(헤드리스에서 빈 화면 방지).

## Bans (지킴)

eyebrow 대문자 라벨, side-stripe, 동일 카드 그리드, 01/02 번호 스캐폴딩, 그라데이션 텍스트, 크림/샌드 배경, 장식용 글래스모피즘.
